# Veni a la fiesta!

**Desarrollo de identidad visual, contenido para redes sociales y sitio web para una marca de invitaciones digitales.**

Cliente: Veni a la fiesta!

**Año:** 2024

**Servicio:** 

Identidad visual

Social Media

Desarrollo web

---

## El proyecto

Vení a la Fiesta! es una marca de invitaciones digitales pensado para simplificar la organización de eventos y ofrecer una alternativa moderna a las invitaciones tradicionales.

La propuesta consiste en invitaciones online personalizadas que permiten compartir toda la información del evento en un solo lugar, facilitando la comunicación con los invitados y aportando una experiencia más práctica y visual.

---

## El desafío

El objetivo del proyecto fue construir una marca atractiva y profesional orientada a eventos como **cumpleaños de 15 y bodas**.

La identidad debía transmitir **delicadeza, modernidad y elegancia**, evitando una estética infantil que pudiera asociarse a invitaciones para cumpleaños de niños. Para lograrlo, se trabajó con una propuesta visual limpia y minimalista, utilizando **colores pasteles y una estética neutra**, donde el protagonismo recae en las fotografías de los eventos y en el diseño de las invitaciones.

El desafío fue crear una identidad visual que acompañara momentos especiales con una estética cuidada y contemporánea.

---

## El enfoque

El trabajo se desarrolló de forma integral, abarcando distintos aspectos de la marca:

- **Identidad visual** con una estética minimalista y moderna
- **Diseño de contenido para redes sociales** para comunicar el servicio y mostrar ejemplos de invitaciones
- **Diseño y desarrollo del sitio web**, donde los usuarios pueden conocer el servicio y explorar las distintas opciones disponibles

Todo el sistema visual fue pensado para mantener coherencia entre la marca, las redes sociales y la experiencia dentro del sitio.

---

## Resultado

El resultado fue una marca digital completa con una identidad visual definida y una presencia online preparada para comunicar el servicio de forma clara y atractiva.

El proyecto permitió desarrollar un sistema visual coherente que funciona tanto en redes sociales como en la web, mostrando de forma clara la propuesta de valor de las invitaciones digitales.
