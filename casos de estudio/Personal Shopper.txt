# Personal Shopper

**Desarrollo de tienda online para una marca que vende productos de moda y skincare**

### Información del cliente

Cliente: Personal Shopper

Año: 2025

Servicios:

Diseño Web

Tienda Online

# El desafío

Florencia llegó a mi con el problema de que gestionaba todas sus ventas manualmente a través de mensajes de Instagram y Whatsapp. Esto implicaba responder consultas de precios, coordinar compras y confirmar pedidos uno por uno.

**Este proceso generaba varios problemas:** consumía mucho tiempo, hacía difícil organizar los pedidos y provocaba que muchas ventas potenciales se perdieran durante la conversación con las clientas.

Además, muchas clientas ya esperaban poder comprar directamente desde una página web, sin tener que gestionar la compra por mensajes.

El objetivo del proyecto fue crear una **tienda online que centralizara las ventas y simplificara el proceso de compra**, tanto para la marca como para sus clientas.

# El enfoque

La solución fue desarrollar un ecommerce completo que permitiera a las clientas navegar los productos, conocer precios y realizar la compra directamente desde la web.

Se diseñó una estructura clara de categorías para organizar el catálogo y facilitar la navegación dentro del sitio, permitiendo encontrar fácilmente los distintos tipos de productos disponibles.

Además, se configuraron todos los aspectos técnicos necesarios para que la tienda funcionara de forma autónoma: métodos de pago, sistema de envíos, gestión de stock y estructura interna del ecommerce.

# La implementación

El sitio fue desarrollado utilizando **WordPress y WooCommerce**, creando una plataforma flexible que la marca pudiera gestionar de forma independiente.

Se realizó la carga inicial de aproximadamente **500 productos**, organizándolos dentro de sus respectivas categorías y optimizando imágenes, títulos y descripciones para mejorar la experiencia de compra.

También se configuró la conexión con la plataforma de pagos y se realizaron pruebas completas del sitio para asegurar que todo el proceso de compra funcionara correctamente antes del lanzamiento.

Una vez finalizado el desarrollo, se brindó una **capacitación para que la clienta pudiera gestionar su tienda**, actualizar productos, controlar el stock y administrar los pedidos desde la plataforma.

# Resultado

El resultado fue una tienda online completamente funcional que permitió profesionalizar el proceso de venta de la marca.

Ahora las clientas pueden navegar el catálogo, ver precios y comprar directamente desde la web, mientras que la marca puede gestionar sus ventas y su stock desde un único lugar, reduciendo el tiempo dedicado a responder consultas y mejorando la organización del negocio.
